Welcome to Thomas' Login System API!

This webpage is simply the greeting page and serves as a README for this API, should you arrive upon it.

Request types for the API

Request Endpoint Requirements Response Description
GET / None Static HTML Serves the greeting page.
GET /login Session: User Authorization Status, User Session Checks if user is already logged in using session storage and cookies.
POST /register Body: Username, Password None Registers a user by inserting row with username and hashed password into MySQL database.
POST /login Body: Username, Password Authorization Status, JWT Token, User Information Checks if user credentials in request match a user in the database using strictly the username, then checks if password matches the hashed password in the database. Upon successful authorization, a user session is saved into the session storage and a JSON Web Token is issued and saved in the local storage.
GET /auth Header: X-Access-Token Authentication Status Checks if the JWT Token matches the one issued to the user.
